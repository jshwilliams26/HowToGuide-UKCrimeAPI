UK Police API Guide

Accessing Stop and Search Information

We will now tie everything together, from forms to GET requests to displaying JSON data, and we will show ways to search for and display Stop and Search Information.

You can specify a location, area, or location id and a date to view stop and searches within an area. Once again, depending on the form information specified, that information will be used to construct a particular query string, and then a GET request will be performed. This JSON data will be parsed and then displayed in a table. Try entering "885142" for the location ID and "2015-06" for the date.

Conclusion

This concludes our How-To-Guide for the UK Police API. You have learned how to navigate JSON data on this API, use forms and JavaScript code to perform GET requests to retrieve this data, and display this data in table format. You can view the documentation and learn more about this API, and try to use these methods to do other requests for this API.
