UK Police API Guide

Introduction

This How To Guide will show you how to pull UK police and crime data from the UK Police API. You can view the documentation at https://data.police.uk/docs/

This Guide will contain JavaScript code examples and explanations for how to pull crime and police information from this database. Because the documentation on the UK API site contains good documentation for what data is pulled from the database, this guide will focus more on writing JavaScript code to access and manipulate this information.

We will retrieve data from the following categories:

  • Police Stations
  • Crime
  • Forces
  • Neighborhoods
  • and more...
