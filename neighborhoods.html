UK Police API Guide

Accessing Neighborhood Information

By using a police force id (in this case, "leicestershire") in the following link https://data.police.uk/api/leicestershire/neighbourhoods, you can view the neighborhoods for that police force. This time, instead of displaying the raw JSON data, we will parse it and display each of the neighborhoods in a table. Enter "leicestershire" as the force id in the form below:

See the Pen gryBjQ by Josh (@willijo4) on CodePen.

We first grab the response text (var response = req.responseText;), then we parse this JSON data (JSON.parse(response)) so that the data could be used as an array of objects. Then this was passed to a function displayNeighborhoods, which contains the appropriate code for a table, and for each object in the array, it will put the id and name of each neighborhood into a row in this table.

You can also enter the ID for both a force and a specific neighborhood to view information for that specific neighborhood. Use "leicestershire" for the force and "NE15" to retrieve the information for that neighborhood. Go to the url https://data.police.uk/api/leicestershire/NE15 to view all of the data for this neighborhood, and edit the CodePen to display other information for this neighborhood. You can view the documentation and explore other options for requesting neighborhood information from the API. We will move on to Stop and Search data, where we will combine what we have learned with creating GET requests and query string with what we have learned about manipulating and displaying JSON data.
